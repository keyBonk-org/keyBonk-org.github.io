## 下标与上标

### 下标

化学式示例：

- 水的分子式是 H~2~O
- 二氧化碳是 CO~2~
- 硫酸的分子式是 H~2~SO~4~

### 上标

数学示例：

- 勾股定理：a^2^ + b^2^ = c^2^
- 爱因斯坦质能方程：E = mc^2^
- 10^3^ = 1000

### 混合使用

有些情况下会同时用到上下标，例如化学中的离子：Ca^2+^、SO~4~^2-^。

## 提示框

### Note 提示

!!! note
    这是一个 note 提示框，用于展示一般性的说明信息。

    提示框可以包含多行内容，支持 **Markdown** 格式。

### Tip 提示

!!! tip
    这是一个 tip 提示框，用于提供实用的小技巧。

    - 技巧一
    - 技巧二

### Info 提示

!!! info
    这是一个 info 提示框，用于传达重要信息。

### Warning 提示

!!! warning
    这是一个 warning 提示框，用于提醒用户注意潜在问题。

    **请务必注意！**

### Danger 提示

!!! danger
    这是一个 danger 提示框，用于警示严重风险。

这是使用 `:::` 语法的自定义容器。
:::
